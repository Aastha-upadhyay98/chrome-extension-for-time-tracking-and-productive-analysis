# Productivity Analytics Chrome Extension

A Chrome extension built to track time spent on various websites and provide productivity insights.

## Features
- **Time Tracking**: Monitors active tab usage in real-time.
- **Backend Storage**: Uses Chrome's storage API to persist user data.
- **Classification**: Automatically categorizes sites like GitHub as "Productive" and social media as "Unproductive".
- **Analytics Dashboard**: Visualizes data via a pie chart to show focus vs. distraction.
- **Weekly Reporting**: Summarizes total time spent over the week.

## How to Install Locally
1. Clone this repository or download the ZIP.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load Unpacked** and select this project folder.
